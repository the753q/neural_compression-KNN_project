## Data:
**Create and manage locally:** \
Download the datasets and extract into the "datasets" directory in the root of the project.

## Should look like:
```
datasets/
├── imagenet_subtrain/
├── DIV2K_train_HR/
├── DIV2K_valid_HR/
├── screenshots/
```

### Imagenet_10k dataset:
https://www.kaggle.com/datasets/priyerana/imagenet-10k

### Div2K
https://www.kaggle.com/datasets/soumikrakshit/div2k-high-resolution-images

### Flickr-Faces-HQ small
https://www.kaggle.com/datasets/tommykamaz/faces-dataset-small

### Minecraft screenshots
https://www.kaggle.com/datasets/sqdartemy/minecraft-screenshots-dataset-with-features
